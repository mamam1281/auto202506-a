- [ ] **무한 스크롤** 또는 **페이지네이션**
- [ ] **정렬 옵션** (인기순, 최신순, 이름순)

3. **사용자 인증** 플로우
4. **지갑 시스템** 기본 기능

### Phase 3: 고급 기능 (2-3주)

2. **프로모션 및 리워드** 시스템
3. **실시간 기능** (채팅, 알림)
