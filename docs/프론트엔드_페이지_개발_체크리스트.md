# 🎯 Casino-Club Frontend 페이지 개발 체크리스트

## 📋 개발 우선순위 및 체크리스트

### 🎪 **1단계: 핵심 게임 페이지 (최우선)**

#### 1.1 메인 대시보드 페이지 (`/dashboard`)
- [ ] **헤더 컴포넌트** (로고, 토큰 잔액, 프로필 메뉴)
- [ ] **게임 카테고리 네비게이션** (슬롯, 테이블, 라이브)
- [ ] **추천 게임 섹션** (인기 게임 카드 그리드)
- [ ] **최근 플레이 게임** (히스토리 카드)
- [ ] **프로모션 배너** (회전식 배너)
- [ ] **푸터 컴포넌트** (링크, 라이센스 정보)

#### 1.2 게임 목록 페이지 (`/games`)
- [ ] **게임 필터링 사이드바** (카테고리, 제공업체, 검색)
- [ ] **게임 카드 그리드** (썸네일, 제목, RTP, 플레이 버튼)
- [ ] **무한 스크롤** 또는 **페이지네이션**
- [ ] **정렬 옵션** (인기순, 최신순, 이름순)
- [ ] **즐겨찾기 기능** (하트 아이콘)

#### 1.3 게임 플레이 페이지 (`/games/[gameId]`)
- [ ] **게임 임베드 영역** (iframe 또는 게임 컨테이너)
- [ ] **게임 컨트롤 패널** (베팅 금액, 스핀 버튼, 자동 플레이)
- [ ] **잔액 표시** (실시간 업데이트)
- [ ] **게임 정보 사이드바** (규칙, RTP, 페이라인)
- [ ] **채팅 위젯** (라이브 게임용)

---

### 🔐 **2단계: 사용자 인증 및 계정 관리**

#### 2.1 로그인/회원가입 페이지 (`/auth`)
- [ ] **로그인 폼** (이메일/사용자명, 비밀번호)
- [ ] **회원가입 폼** (단계별 폼 또는 통합 폼)
- [ ] **소셜 로그인** (Google, Facebook 버튼)
- [ ] **비밀번호 재설정** (이메일 입력 폼)
- [ ] **이용약관 동의** (체크박스, 모달)

#### 2.2 마이페이지 (`/profile`)
- [ ] **프로필 정보 탭** (기본 정보 수정)
- [ ] **보안 설정 탭** (비밀번호 변경, 2FA)
- [ ] **게임 히스토리 탭** (플레이 기록, 승패 통계)
- [ ] **거래 내역 탭** (입출금 기록)
- [ ] **즐겨찾기 게임 탭** (북마크한 게임들)

---

### 💰 **3단계: 결제 및 지갑 시스템**

#### 3.1 지갑 페이지 (`/wallet`)
- [ ] **잔액 대시보드** (총 잔액, 보너스 잔액, 현금 잔액)
- [ ] **입금 섹션** (결제 수단 선택, 금액 입력)
- [ ] **출금 섹션** (출금 계좌 설정, 출금 신청)
- [ ] **거래 내역 테이블** (날짜, 유형, 금액, 상태)
- [ ] **보너스 관리** (활성 보너스, 조건 진행률)

#### 3.2 결제 페이지 (`/payment`)
- [ ] **결제 수단 선택** (카드, 전자지갑, 암호화폐)
- [ ] **결제 정보 입력 폼** (안전한 입력 필드)
- [ ] **결제 확인 및 처리** (로딩 상태, 결과 표시)
- [ ] **영수증 및 확인서** (다운로드 가능한 PDF)

## 🛠 **기술적 구현 순서**

2. **라우팅 구조** 설정 (Next.js App Router 활용)
3. **전역 상태 관리** 설정 (사용자 정보, 게임 상태)
4. **API 통신 레이어** 구축 (fetch 래퍼, 에러 핸들링)

### Phase 2: 핵심 기능 (2-3주)
1. **메인 대시보드** 구현
2. **게임 목록 및 상세** 페이지
3. **사용자 인증** 플로우
4. **지갑 시스템** 기본 기능

### Phase 3: 고급 기능 (2-3주)
1. **결제 시스템** 통합
2. **프로모션 및 리워드** 시스템
3. **실시간 기능** (채팅, 알림)
4. **통계 및 분석** 페이지

### Phase 4: 최적화 및 완성 (1-2주)
1. **성능 최적화** (이미지, 번들 크기, 로딩 속도)

3. **접근성** 개선 (a11y)
4. **SEO 최적화** (메타태그, 구조화된 데이터)
